Mapping Portfolio

The Plaza Coral Gables

The goal of this project is to provide insights into market demographic data for the area surrounding a new commercial real estate development, The Plaza Coral Gables. Population density is displayed for portions of Miami, FL surrounding the development.

Map prepared with QGIS 3.4.4 on macOS 10. Rule based symbology was used to create map design elements. Classes were created using Jenks natural breaks classification.

Data sources:

  • US Census Bureau
  • American Fact Finder

The Plaza Coral Gables

In this second map for The Plaza Coral Gables, higher education resources are displayed within concentric 3 mile buffers surrounding the development.

Buffer geoprocessing was utilized to created the concentric ring buffers.

Data source:

  • University of Florida GeoPlan Center

Nolichucky River

Running between North Carolina and East Tennessee, the Nolichucky River provides one of the most techinical sections of whitewater paddling in the the southern US. This map identifies the 14 major rapids along the whitewater portion of the river, along with their classifications.

Data prepared with QGIS 3.4.4 on macOS 10. Web map created with Leaflet.

Data sources:

  • A Wet State
  • American Whitewater
  • Esri Basemaps

The Appalachian Trail

Stretching more than 2,000 miles, the Appalachian Trail (AT) is one of the worldâ€™s longest footpaths. This map displays the portion of the AT that runs along the Smoky Mountains and the Blue Ridge Mountains in Tennessee, North Carolina, & Virginia.

The base map was created from a hand draw, Erwin Raisz Landform Map. The Raisz map was georeferenced to a 1:10m graticule provided by Natural Earth.

Breweries in North America

This map was created as a study in location analysis for breweries across North America and Hawaii. Brewery density is displayed in a 50km hex grid.

Map prepared with QGIS 3.4.4 on macOS 10. Hex bin analysis was performed to display density.

Data sources:

  • POI Factory
  • Natural Earth
  • US Census Bureau

Breweries in Chicago, IL

Chicago, IL is one of the top-ten highest density areas for breweries in North America. This map displays brewery density for Chicago in a 2km hex grid.

Map prepared with QGIS 3.4.4 on macOS 10. Hex bin analysis was performed to display density.

Data sources:

  • POI Factory
  • Natural Earth
